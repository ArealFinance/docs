Areal использует **мульти-токен архитектуру**, где каждый токен имеет чётко определённую роль, область применения и экономическую функцию. Ни один токен не призван решать все задачи системы — вместо этого токены дополняют друг друга на разных уровнях.

<CardGroup cols={3}>
  <Card title="RWT" icon="coins" color="#a56eff" href="/ru/economics/rwt-real-world-token">
    Базовый токен доходности и ликвидности — агрегирует реальную доходность в один унифицированный актив
  </Card>
  <Card title="Ownership Tokens" icon="key" color="#10b981" href="/ru/economics/ownership-tokens">
    Проектные токены владения — представляют реальные, юридически подкреплённые права на RWA-проекты
  </Card>
  <Card title="ARL" icon="crown" color="#3b82f6" href="/ru/economics/arl-protocol-token">
    Governance-токен протокола — управляет Areal через futarchy-модель принятия решений
  </Card>
</CardGroup>

<Info>
  Каждый токен работает на своём уровне системы. Вместе они формируют замкнутый экономический цикл: Ownership Tokens генерируют доходность от реальных активов → RWT агрегирует эту доходность в единую ликвидность → ARL управляет правилами, которые связывают всё воедино.
</Info>

## RWT — Real World Token

<Tooltip tip="Real World Token — основной утилитарный токен протокола Areal для унифицированной ликвидности и агрегации доходности">**RWT**</Tooltip> — базовый токен протокола, созданный для решения проблемы **фрагментированной ликвидности** на рынке RWA.

Вместо распыления капитала по десяткам изолированных пулов, Areal агрегирует доходность от множества реальных активов в один токен. RWT накапливает диверсифицированную RWA-доходность через <Tooltip tip="Ownership Token — токенизированное представление конкретного реального актива в протоколе Areal">Ownership Tokens</Tooltip>, хранящиеся в его vault, становясь единой точкой доступа ко всему портфелю протокола.

## Ownership Tokens

<Tooltip tip="Ownership Token — представляет реальное, юридически закреплённое владение конкретным реальным активом в Areal">**Ownership Tokens (OTs)**</Tooltip> представляют **реальное, юридически закреплённое владение** материальными и нематериальными активами в экосистеме Areal.

Каждый Ownership Token привязан к **DAO Ownership Company** — юридическому лицу, которое официально владеет всеми активами проекта. Держатели токенов имеют реальные права на реальную стоимость, а не просто спекулятивный цифровой актив.

  </Card>
  <Card title="Свободно торгуемый" icon="arrow-right-arrow-left">
    OT свободно торгуются на открытом рынке, обеспечивая ликвидность и прозрачное ценообразование для каждого проекта

## ARL — Токен протокола

<Tooltip tip="ARL — governance- и ownership-токен протокола Areal, подкреплённый DAO Ownership Company">**ARL**</Tooltip> — это <Tooltip tip="Ownership Token — представляет реальное, юридически закреплённое владение конкретным реальным активом в Areal">Ownership Token</Tooltip> **самого протокола Areal**.

Как любой Ownership Token представляет реальное владение конкретным RWA-проектом, так ARL представляет владение протоколом Areal — включая всю интеллектуальную собственность, продукты, инфраструктуру и экономические потоки. Держатели ARL коллективно управляют протоколом через [futarchy](/ru/architecture/governance-and-futarchy).
