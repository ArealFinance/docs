# Areal Finance — The RWA DeFi Hub

**Areal Finance is a full-stack onchain protocol for launching, owning, and participating in real-world assets.**

Instead of fragmenting liquidity across dozens of isolated tokens, Areal aggregates all RWA activity into a single architecture — one shared liquidity layer, one yield distribution model, and one governance system. The result is a unified DeFi hub where real-world assets become composable, liquid, and collectively governed.
